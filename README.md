# Roadmap Time Visão - Machine Learning
_Bem-vindo ao repositório do site do Roadmap da área de Machine para quem está interessado em começar na área ou buscar algum tópico de interesse com sugestões de livros, playlists, ou qualquer tipo de recurso para auxiliar o seu estudo. Este repositório foi uma iniciativa do [Time Visão](https://timevisaoupe.github.io/) para auxiliar quem gostaria de entrar na área._

## Conteúdos abordados no Roadmap
O roadmap aborda os seguintes conteúdos da área de ML:

 - Áreas do conhecimento matemático para Aprendizado de Máquina e Dados;
 - Conhecimento de programação, desde as bibliotécas de manipulação, visualização e análise de dados, até as específicas de ML e áreas correlatas (ex. Visão Computacional);
 - Conhecimento da área de dados e de como manipular os dados de forma a criar um modelo limpo e eficiente
 - Tópicos de Machine Learning e Inteligência Artificial, abordando aspéctos teóricos e práticos para o treinamento de um modelo.
----
## O que foi utilizado para fazer o site:
No desenvolvimento do site foi utilizado apenas o SSG [Vitepress](https://vitepress.dev/pt/), para fazer um site estático estilo Wiki com uma interface gráfica melhor para esse tipo de documentação.

----
## Deploy
O site foi hospeadado no [Vercel](https://vercel.com/). Toda modificação na branch `Main` vai ser automaticamente publicada no site.
